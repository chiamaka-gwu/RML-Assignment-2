# Individual Homework 2 - COMPAS Explainability

## Overview
This project builds on Lecture 2 and explains a COMPAS replacement model using SHAP, LIME, and counterfactual explanations.

## How to Run
1. Open the notebook
2. Run all cells from top to bottom
3. The dataset is loaded automatically from GitHub

## Notes
The goal of this assignment is to analyze model behavior and highlight limitations of explanation methods in a governance context.
